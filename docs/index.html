기능

한국어 초안에서 발행된 Tistory 글까지

글을 패키지로 묶고, 고유 커버를 만들고, 내 브라우저 세션을 그대로 써서 비공개로 저장하는 전 과정을 스킬이 담고 있습니다.

01

비공개 기본 발행

명시적으로 확인하기 전에는 공개로 발행하지 않습니다. 기본은 언제나 비공개 또는 초안입니다.

02

마크다운을 패키지로

마크다운 또는 Hugo 글을 바로 올릴 수 있는 Tistory 패키지로 변환합니다.

03

커버 프롬프트 자동 생성

제목, 요약, 태그, heading, 본문 신호로 cover_prompt.md를 만들어 둡니다.

04

토픽 맞춤 Codex 커버

글마다 고유한 Codex Image 2.0 커버를 요구합니다. 재활용 배너는 거부합니다.

05

읽기 좋은 본문 간격

문단, heading, 목록, 표, 인용, 코드에 깔끔한 Tistory 본문 간격을 적용합니다.

06

내 Chrome 세션 재사용

이미 로그인된 Chrome에 Playwright CLI를 CDP로 붙여 그대로 작업합니다. 쿠키를 따로 내보내지 않습니다.

작동 방식

여섯 단계, 한 세션

글쓰기가 먼저입니다. 발행은 글이 준비된 다음 돌아가는 결정적 파이프라인입니다.

1

작성

작성 표준에 맞춰 한국어 SEO 글을 마크다운으로 씁니다.

2

패키지

prepare_tistory_package.py가 패키지와 토픽 기반 cover_prompt.md를 만듭니다.

3

커버

Codex Image 2.0으로 고유 커버를 만들어 cover.codex.png로 저장합니다.

4

연결

Playwright CLI를 로그인된 Chrome에 CDP로 붙입니다.

5

렌더

커버를 올리고 읽기 좋은 한국어 본문 HTML을 렌더합니다.

6

발행

비공개로 저장합니다. 공개 전환은 사용자가 명시적으로 확인할 때만 합니다.

빠른 시작

설치하고 바로 실행

Claude Code 또는 Codex에 스킬을 연결하고, 글을 패키지로 묶은 뒤 비공개로 발행합니다.

1 설치 (standalone 클론 링크)

# Claude Code
ln -s "$PWD" ~/.claude/skills/tistory-seo-blog-poster
# Codex
ln -s "$PWD" ~/.codex/skills/tistory-seo-blog-poster

2 패키지 준비

python3 scripts/prepare_tistory_package.py \
  --source /path/to/post.md \
  --output-dir /tmp/tistory-post

3 비공개 발행 (커버 생성, Chrome 연결 후)

python3 scripts/tistory_playwright_cli.py \
  --package /tmp/tistory-post/package.json \
  --session tistory-cdp --attach none \
  --target-url "https://YOUR_BLOG.tistory.com/manage/newpost/?type=post" \
  --private-publish --confirm-private-publish

가드레일

설계부터 안전하게

계정을 보호하고 커버를 진짜로 유지하도록 만들어졌습니다.

커버는 글마다 하나. 재활용이나 일반 AI 배너는 업로드 전에 거부합니다.
최종 발행은 진짜 커버만. Codex가 가능할 때 로컬 폴백 커버는 쓰지 않습니다.
로그인은 사용자 브라우저에. 쿠키나 프로필을 repo로 내보내지 않습니다.
비공개와 초안 우선. 공개 발행은 명시적으로 요청할 때만 합니다.
